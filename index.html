2. Three Experiments

Exp 1 — Custom CNN

  • Built from scratch (shallow)
  • 4 conv blocks: 16→32→64→128 filters
  • BatchNorm + ReLU + MaxPool
  • Dropout 50%
  • Adam optimiser, LR 0.001
  • 20 epochs, mini-batch 32

Exp 2 — SqueezeNet

  • Pretrained on ImageNet (1.2M images)
  • Fire modules (squeeze + expand)
  • Only 1.2M parameters
  • Removed last 5 layers → added Conv(1×1,2)
  • New layers learn 10× faster
  • SGDM, LR 0.0001, 15 epochs

Exp 3 — ResNet-18

  • Pretrained on ImageNet (1.2M images)
  • Skip (residual) connections
  • 18 layers, ~11M parameters
  • Removed last 3 layers → added FC(2)
  • New layers learn 10× faster
  • SGDM, LR 0.0001, 15 epochs

4. Key Decisions

Image Size
224×224 / 227×227
224 for CNN & ResNet, 227 for SqueezeNet
Oversampling
TB 490 → 2,450
Fix 5:1 class imbalance
Data Split
70 / 15 / 15
Separate test set for fair evaluation
Transfer LR
0.0001
Preserves pretrained features
Augmentation
Rotation + Flip + Shift
Prevents overfitting
Metrics
F1 + ROC/AUC
Accuracy alone is misleading for imbalanced data
